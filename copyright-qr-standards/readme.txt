Automatically adds a copyright notice and QR Code to the bottom of your single posts.

== Description ==

This plugin automatically adds two elements to the bottom of your single blog posts:
1. A copyright notice including your site name and the current year.
2. A QR Code that links to the current post URL.

Features:
* Fully compliant with WordPress Coding Standards.
* Internationalization (i18n) ready.
* Responsive design.
* Lightweight and fast.
* No settings required - works out of the box.

== Installation ==

1. Upload the `copyright-qr-standards` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Visit any single post to see the result.
